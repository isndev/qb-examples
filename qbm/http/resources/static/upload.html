API Endpoints

Direct API Endpoints

GET /api/files - List all uploaded files (JSON)
POST /api/upload - Upload endpoint (used by form above)
GET /browse - Browse uploaded files (HTML)
DELETE /api/files/:filename - Delete uploaded file

cURL Examples

# Upload a file with description and tags
curl -X POST \
  -F "file=@yourfile.txt" \
  -F "description=Test file upload" \
  -F "tags=test,example,demo" \
  http://localhost:8080/api/upload

# List all uploaded files
curl http://localhost:8080/api/files

# Get specific file metadata
curl http://localhost:8080/api/files/upload_1234567890_yourfile.txt

# Delete a file
curl -X DELETE http://localhost:8080/api/files/upload_1234567890_yourfile.txt
