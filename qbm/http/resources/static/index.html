QB HTTP Static File Server Example

Welcome to the QB HTTP framework static file server demonstration.

Features Demonstrated

  • Static file serving with proper MIME types
  • Directory browsing and file listing
  • File upload via multipart/form-data
  • JSON API for file operations
  • Caching headers and ETag support
  • Range requests for large files
  • Security measures against path traversal

API Endpoints

  • GET /api/files - List all uploaded files
  • POST /api/upload - Upload a new file
  • GET /api/files/:filename - Get file metadata
  • DELETE /api/files/:filename - Delete a file
  • PUT /api/files/:filename/metadata - Update file metadata
